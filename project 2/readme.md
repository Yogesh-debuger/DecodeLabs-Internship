# 🚀 Project 2: The Server Commander

This project demonstrates how to set up a cloud server and host a webpage on it.

As part of this project, I performed the following tasks:

* Launched a **Linux EC2 instance** on AWS
* Connected to the server securely using **SSH**
* Installed **Nginx Web Server** using the terminal
* Configured the server to run the web service
* Created and hosted a **custom "Welcome to DecodeLabs" webpage**
* Accessed the webpage using the **public IP of the server**

This project helped me understand the basics of **cloud servers, Linux administration, and web hosting**.

---

## 🛠 Tools Used

* AWS EC2
* Linux
* Nginx
* SSH
* Terminal

---

## 🌐 Output

The custom webpage is successfully hosted on the cloud server and accessible through the server’s **public IP address**.
